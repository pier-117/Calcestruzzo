ISTRUZIONI:

1. Installa le librerie:
   pip install -r requirements.txt

2. Inserisci il tuo token nel file bot.py alla riga app = Application.builder().token("...")

3. Avvia il bot:
   python bot.py

✅ Accesso solo con PIN (es: 1234, 5678)
✅ Firma compilata automaticamente in base all'utente loggato
✅ Salvataggio su Excel

Comandi:
- /start → avvia login e compilazione
- /annulla → annulla operazione corrente
